# IRIS-Logbook

IRIS-Logbook adalah short-term project dari IRIS FTMM Universitas Airlangga yang berfokus pada penerapan image processing untuk membangun sistem absensi berbasis pengenalan wajah.

Project ini dikembangkan dalam bentuk web app, sehingga proses absensi dapat dilakukan secara lebih praktis, otomatis, dan real-time melalui kamera.

## Tujuan Project
- Mengembangkan sistem absensi otomatis berbasis face recognition
- Menerapkan metode image processing dan deep learning dalam skenario nyata
- Menyediakan web app sederhana untuk proses absensi

## Cakupan Sistem
- Ekstraksi wajah dari data video
- Pelatihan model klasifikasi wajah
- Inferensi wajah secara real-time melalui webcam
- Pencatatan kehadiran ke dalam log absensi

## Tech Stack
- Python
- OpenCV
- Flask
- ResNet-50
- MTCNN

## Output
Web app absensi berbasis face recognition yang dapat mengenali pengguna dan mencatat kehadiran secara otomatis.
